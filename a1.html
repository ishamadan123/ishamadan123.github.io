Assignment 1: Blink

For this assignment, we were tasked with creating a schematic for at least 3 LEDs connected to the Arduino, which I then implemented as a circuit on my breadboard.

I used the following calculation to decide the resistors to use: V(Volts) = I (Amperes) * R (Ohms)

The voltage of the Arduino is 5V, and each LED light I chose to use (red, yellow, and red) each had a voltage drop of 1.8V and a desired current of 20mA. Since this circuit is in parallel, we only need to use to voltage of a single LED.

(5-1.8) = 0.02A * R
3.2 = 0.02 * R
R = 3.2 / 0.02
R = 160 Ohms

I chose to use three 220 Ohms resistor instead of 160, because it just had to be greater than 150, and might effect the brightness of the light, but would still function.

          Turns on first LED on for half a  second, then off for half a second, then the second LED turns

        // the setup function runs once when you press reset or power the board
        void setup() {
          // initialize digital pins 8, 10, and 12, as an output, because my wires connected to those pins on the arduino
          pinMode(8, OUTPUT);
          pinMode(10, OUTPUT);
          pinMode(12, OUTPUT);
        }
        
        // the loop function runs over and over again forever
        void loop() {
          digitalWrite(8, HIGH);  // turn the first LED on (HIGH is the voltage level)
          delay(15);                      // wait for a half a second
          digitalWrite(8, LOW);   // turn the first LED off by making the voltage LOW
          delay(15);                      // wait for half a second
        
          digitalWrite(10, HIGH);  // turn the second LED on (HIGH is the voltage level)
          delay(15);                      // wait for a half a second
          digitalWrite(10, LOW);   // turn the second LED off by making the voltage LOW
          delay(15);                      // wait for a half a second
        
          digitalWrite(12, HIGH);  // turn the third LED on (HIGH is the voltage level)
          delay(15);                      // wait for a half a second
          digitalWrite(12, LOW);   // turn the third LED off by making the voltage LOW
          delay(15);                      // wait for a half a second
        }
